# app-policies

宇佐見惇が提供するモバイルアプリケーションのプライバシーポリシー・サポートページを公開するリポジトリ。

GitHub Pages で公開中： https://nujimasu.github.io/app-policies/

## 構成

```
.
├── index.md                  # ランディングページ
├── _config.yml               # Jekyll 設定
├── mobihub/
│   ├── privacy-policy.md
│   └── support.md
└── driveq/
    ├── privacy-policy.md
    └── support.md
```

## 更新方法

Markdown ファイルを編集して push すれば、数分後に GitHub Pages に反映される。
